Gp1 Lap Edit

Description Gp1 lap edit is a lap editor for Grand Prix 1. With this tool you can change the Nr of laps in a 100% race between 3 and 127 laps. This tool is written in Visual C++ and you don't need any runtime files!
New in v1.1 is a much needed design update and built in checksum function.
